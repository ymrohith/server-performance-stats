# server-performance-stats
Script to analyse server performance stats using Bash

## Features:
- CPU usage
- Memory usage
- Disk usage
- Top 5 CPU/memory processes
- Bonus: OS info, uptime, load average

## How to Run:
```bash
chmod +x server-stats.sh
./server-stats.sh
